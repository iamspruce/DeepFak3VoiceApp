Your computer will show a security warning because we're an independent developer. This is completely normal and safe!

Step 0: Extract the Download

  1. Find the DeepFak3rVoiceVibe.zip file you downloaded from Gumroad
  2. Right-click the zip file and choose "Extract All"
  3. Pick a location like your Desktop or Documents folder
  4. Inside the extracted folder, you'll see:
    • DeepFak3rVoice.exe
    • certificate.pfx or certificate.cer (security certificate)
    • Setup guide PDF

Step 1: Install the Certificate (One-Time Setup)

  1. Find the certificate file in the extracted folder (named certificate.pfx or certificate.cer)
  2. Right-click the certificate file
  3. Choose "Install Certificate"
  4. A window appears:
    • Select "Local Machine"
    • Click "Next"
    • Click "Yes" when Windows asks for permission
  5. Choose "Place all certificates in the following store"
  6. Click "Browse"
  7. Select "Trusted Root Certification Authorities"
  8. Click "OK", then "Next", then "Finish"
  9. Click "Yes" when Windows asks "Do you want to install this certificate?"
  10. You'll see "The import was successful" ‚Äì click "OK"

Step 2: Run the App

  1. Double-click DeepFak3rVoice.exe in the folder
  2. If Windows Defender shows a warning:
    • Click "More info"
    • Click "Run anyway"
  3. The app will open!

Step 3: Enter Your License Key

  1. The app will show a license activation screen
  2. Open your Gumroad purchase email
  3. Copy the license key (a long string of letters and numbers)
  4. Paste it into the app
  5. Click "Activate"
