Mac is very protective about apps from independent developers. Follow these steps carefully.

Step 0: Extract the Download

  1. Find the DeepFak3rVoiceVibe.zip file in your Downloads folder
  2. Double-click to extract it
  3. You'll see a folder with DeepFak3rVoice-universal.dmg inside

Step 1: Install the App

  1. Double-click the DeepFak3rVoice-universal.dmg file
  2. A window opens showing the DeepFak3r Vibe Voice app icon
  3. Drag the app to your Applications folder (or Desktop)
  4. Eject the DMG by right-clicking it and choosing "Eject"

Step 2: Trust the App (One-Time Setup)

Mac will block the app the first time. Here's how to allow it:

Method 1: Right-Click to Open (Try This First)

  1. Go to where you installed the app (Applications folder or Desktop)
  2. Right-click (or Control-click) the DeepFak3r Vibe Voice app
  3. Choose "Open" from the menu
  4. A warning appears ‚Äì click "Open" again
  5. If this works, skip to Step 3!

Method 2: If Mac Completely Blocks It

  1. Open System Settings (click the Apple menu at top left)
  2. Go to "Privacy & Security"
  3. Scroll down to the Security section
  4. You should see a message like: "DeepFak3rVoice was blocked from use because it is not from an identified developer"
  5. Click "Open Anyway" next to that message
  6. Click "Open" in the popup that appears
  7. The app will now start!

Method 3: Using Terminal (For Advanced Users)

If the above doesn't work:

  1. Open Terminal (search "Terminal" in Spotlight ‚Äì press Command + Space)
  2. Type this command but don't press Enter yet: 
    sudo spctl --add
  3. Add a space after --add
  4. Drag the DeepFak3r Vibe Voice app icon from Finder into the Terminal window (this adds the path automatically)
  5. Press Enter
  6. Type your Mac password when asked (you won't see it as you type ‚Äì that's normal)
  7. Press Enter again
  8. Close Terminal and try opening the app normally

Step 3: Enter Your License Key

  1. The app will show a license activation screen
  2. Open your Gumroad purchase email
  3. Copy the license key
  4. Paste it into the app
  5. Click "Activate"
