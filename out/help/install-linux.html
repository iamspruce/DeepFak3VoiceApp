üêß Installing on Linux

Works on Ubuntu, Fedora, Arch, and most other distributions.

Step 1: Extract the App

Using File Manager:

  1. Find the DeepFak3rVoice-Linux.tar.gz file in your downloads
  2. Right-click and choose "Extract Here"
  3. You'll see a folder with the DeepFak3rVoice file inside

Using Terminal:

cd ~/Downloads
tar -xzf DeepFak3rVoice-Linux.tar.gz
cd DeepFak3rVoice

Step 2: Make It Executable

Using File Manager:

  1. Right-click the DeepFak3rVoice file
  2. Choose "Properties"
  3. Go to the "Permissions" tab
  4. Check the box: "Allow executing file as program"
  5. Close the window

Using Terminal:

chmod +x DeepFak3rVoice

Step 3: Run the App

Using File Manager:

Double-click the DeepFak3rVoice file

Using Terminal:

./DeepFak3rVoice

Step 4: Enter Your License Key

  1. The app shows a license activation screen
  2. Open your Gumroad purchase email
  3. Copy the license key
  4. Paste it into the app
  5. Click "Activate"
